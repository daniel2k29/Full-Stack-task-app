# Full-Stack Task Management App

## Overview
A full-stack task management web application built with React (Vite + TypeScript) and Node.js (Express + TypeScript), using PostgreSQL as the database and Prisma as the ORM. The app is Dockerized and has CI/CD configured.

---

## Features
- User authentication (JWT)
- Task CRUD (Create, Read, Update, Delete)
- Responsive UI with TailwindCSS
- Dockerized backend and frontend
- CI/CD with GitHub Actions

---
